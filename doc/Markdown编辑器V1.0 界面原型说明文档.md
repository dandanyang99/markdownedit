# 二、整体视觉规范

## 2.1 色彩规范

- 主色调：蓝色（#4285F4），用于激活状态、按钮、核心交互元素，传递专业、简洁的视觉感受

- 中性色：
        

    - 背景色（浅色模式）：中性灰（#F5F5F5）、纯白（#FFFFFF）；专注模式背景（#F9F9F9）

    - 背景色（深色模式）：深灰（#1E1E1E）、纯黑（#1A1A1A）；专注模式背景（#1A1A1A）

    - 文字色（浅色模式）：深灰（#333333）、中灰（#666666）；选中高亮（#B3D4FC）

    - 文字色（深色模式）：浅灰（#E0E0E0）、中灰（#999999）；选中高亮（#4A6FA5）

    - 分隔线/边框色：浅灰（#E0E0E0），用于区域划分、组件边框

- 提示色：
        

    - 错误色：红色（#FF4444），用于错误提示、异常状态

    - 警告色：橙色（#FFAA00），用于警告提示、待处理状态

    - 成功色：绿色（#00C853），用于成功提示、完成状态（同步完成、导出成功）

## 2.2 尺寸与形状规范

- 圆角：所有界面元素统一使用4px圆角（功能弹窗使用8px圆角），避免尖锐边角，提升视觉柔和度

- 间距：基础间距8px，功能区间距16px，编辑区域留白20px（专注模式），确保界面呼吸感均衡

- 图标尺寸：工具栏功能图标16x16px，模式切换/主题图标18x18px，Logo图标24x24px，保持视觉一致性

- 字体规范：
        

    - 默认字体：Windows（微软雅黑）、macOS（苹方），统一为无衬线字体，提升可读性

    - 字号层级：标题（弹窗标题14px加粗）、正文（编辑区域14px）、辅助文字（状态栏12px）

    - 行高：编辑区域行高1.6倍，段落间距12px，确保长文本阅读流畅

# 三、核心界面布局与细节说明

整体布局分为五大区域：顶部工具栏、左侧侧边栏、中间编辑区域、右侧预览区（分栏模式专属）、底部状态栏，支持自定义隐藏/显示部分区域，记忆用户布局偏好。

## 3.1 顶部工具栏（高度固定50px）

贯穿整个窗口顶部，包含文件操作、格式工具、模式切换、设置入口，功能分区明确，操作便捷。

- 左侧区域（文件操作）：
        

    - 产品Logo：居左摆放，尺寸24x24px，点击可返回默认工作区

    - 文件下拉菜单：包含新建、打开、保存、导出选项，文字色#333333，hover时背景色#E8F0FE，无额外边框，点击后展开下拉列表（列表项高度32px）

- 中间区域（格式工具）：
        

    - 按功能分组排列，图标+文字组合展示，图标16x16px，组间间距16px，项间间距8px

    - 分组逻辑：文本格式（加粗、斜体、删除线）→ 段落格式（标题、列表、引用）→ 插入功能（链接、图片、表格、公式）

    - 状态样式：未激活状态文字/图标色#666666，hover/激活状态变为#4285F4；点击时有轻微按压反馈（背景色#E0E0E0，按压时视觉下沉1px）

- 右侧区域（模式与设置）：
       

    - 模式切换按钮：所见即所得、源码、分栏预览三个选项，图标+文字组合，选中状态下方显示2px高#4285F4下边框，未选中状态文字色#666666

    - 主题切换：明暗模式切换图标（18x18px），点击一键切换，记忆用户偏好

    - 设置入口：齿轮图标（18x18px），点击弹出下拉菜单（包含偏好设置、快捷键、帮助选项），菜单项hover背景色#E8F0FE

## 3.2 左侧侧边栏（默认宽度220px，可折叠）

位于窗口左侧，支持折叠至仅显示图标（折叠后宽度40px），包含文件列表与大纲视图两大功能，可通过顶部标签切换。

- 顶部切换标签（高度40px）：
        

    - 两个标签（文件列表、大纲视图）均分侧边栏宽度，文字色#333333，选中状态背景色#E8F0FE、文字色#4285F4，无额外边框，点击切换视图

- 文件列表视图：
        

    - 按文件夹层级展示，文件夹前有展开/折叠箭头（默认折叠，箭头方向垂直/水平切换）

    - 图标区分：.md文件使用专属Markdown图标，文件夹使用系统风格文件夹图标，图标尺寸16x16px，与文字左对齐，间距4px

    - 交互样式：选中文件背景色#E0E0E0，文字色#333333；右键弹出文件操作菜单（重命名、删除、复制路径等），菜单项高度32px；支持拖拽调整文件/文件夹位置，拖拽时显示半透明占位符

- 大纲视图：
        

    - 按标题层级缩进展示，H1无缩进，H2缩进12px，H3缩进24px，以此类推，层级清晰

    - 高亮规则：当前编辑位置对应的标题加粗、文字色#4285F4，其他标题文字色#333333

    - 交互能力：点击大纲项可快速跳转至文档对应段落；标题前有折叠箭头，支持单独折叠/展开该层级下内容

- 侧边栏底部（添加按钮）：
       

    - +图标（18x18px）+“新建”文字组合，居左摆放，背景色#F5F5F5，hover时变为#E8F0FE

    - 点击弹出选项（新建文件、新建文件夹），选项项高32px，hover背景色#E8F0FE

## 3.3 中间编辑区域（核心区域）

位于界面中央，随窗口大小自适应调整，支持所见即所得、源码、分栏预览、专注四种模式，是核心写作区域。

- 基础样式（所见即所得模式，默认）：
        

    - 无多余边框，背景色随主题切换（浅色#FFFFFF，深色#1E1E1E），文字色对应为#333333、#E0E0E0

    - 光标样式：竖线光标，宽度2px，颜色#4285F4，闪烁频率1次/秒；选中内容背景色#B3D4FC（浅色）、#4A6FA5（深色），选中文字色不变

    - 内容排版：默认字号14px，行高1.6倍，段落间距12px，左右留白16px，避免内容贴边

- 分栏模式适配：

    - 中间显示1px宽#E0E0E0分隔线，拖拽分隔线可调整分栏比例，拖拽时分隔线变为2px宽#4285F4

    - 限制规则：左侧编辑区最小宽度180px，右侧预览区最小宽度200px，避免因宽度过窄导致内容错乱

- 专注模式样式：
        

    - 隐藏工具栏、侧边栏、状态栏，仅保留编辑区域，四周留白20px，提升沉浸感

    - 背景色可自定义，默认浅色#F9F9F9、深色#1A1A1A，支持通过设置调整

    - 快捷键触发（Ctrl+/），再次触发退出专注模式，记忆用户专注模式下的背景色偏好

- 源码模式样式：
        

    - 纯文本展示，Markdown语法符号高亮（标题#、列表符号、链接括号等），高亮色#4285F4

    - 排版与所见即所得模式一致，保证切换模式后内容位置无偏移

## 3.4 右侧预览区（分栏模式专属）

仅在分栏预览模式下显示，位于编辑区域右侧，实时渲染文档格式，与左侧编辑内容同步联动。

- 基础布局：与中间编辑区域保持一致，左右留白16px，背景色、文字色随主题切换，与编辑区域风格统一

- 同步规则：鼠标滚轮滚动时，与左侧编辑区域内容同步（上下偏差≤3行），确保编辑与预览位置对应

- 内容样式：
        

    - 标题层级：H1（24px、加粗）、H2（20px、加粗）、H3（18px、加粗）、H4-H6（16px、加粗），逐级递减，与编辑区域格式统一

    - 代码块：背景色#F5F5F5（浅色）、#2D2D2D（深色），边框-radius 4px，内边距8px，字体为等宽字体（Consolas）

    - 表格：边框1px #E0E0E0，表头背景色#F5F5F5（浅色模式）、#333333（深色模式），单元格内边距6px

    - 公式与图表：实时渲染，居中显示，与正文间距12px，确保渲染清晰无模糊

## 3.5 底部状态栏（高度固定24px）

位于窗口底部，展示核心状态信息，样式简洁，不遮挡编辑内容，支持hover查看详情。

- 左侧区域：
        

    - 字数统计：格式为“字数：XXX 字符数：XXX”，文字色#666666，字号12px

    - 模式提示：显示当前编辑模式（如“所见即所得模式”），文字色#666666，字号12px，与字数统计间距16px

- 中间区域（同步状态）：
        

    - 图标+文字组合，图标12x12px，文字色#666666，字号12px

    - 状态展示：同步中（旋转加载图标+“同步中...”）、同步完成（绿色对勾图标+“同步完成”）、同步失败（红色警告图标+“同步失败”）

    - 交互：hover时显示详细信息（如同步时间、失败原因），同步失败时点击可触发重试

- 右侧区域：
        

    - 格式提示：如“已兼容GFM语法”，文字色#666666，字号12px

    - 版本信息：如“V1.0.0”，文字色#666666，字号12px，与格式提示间距16px

# 四、弹窗与提示样式说明

## 4.1 轻量操作提示（自动消失型）

- 适用场景：文件保存、同步完成、导出成功等基础操作反馈

- 样式：居下居中显示，背景色#333333（半透明80%），文字色#FFFFFF，字号12px，内边距8px 16px，边框-radius 4px

- 交互：自动显示2秒后消失，不遮挡编辑区域，无手动关闭按钮，同一时间仅显示一个提示

## 4.2 警告/错误提示（带操作按钮型）

- 适用场景：格式错误、同步失败、权限不足等需要用户关注或操作的场景

- 样式：居下居中显示，错误背景色#FF4444、警告背景色#FFAA00，文字色#FFFFFF，字号12px，内边距8px 16px，边框-radius 4px

- 交互：附带操作按钮（如“重试”“查看详情”），按钮文字色#FFFFFF，背景色透明，边框1px solid #FFFFFF，点击区域≥40px宽；点击按钮触发对应操作，点击提示外区域可关闭提示

## 4.3 功能弹窗（如导出设置、分享设置）

- 样式：

    - 位置：居中显示，宽度默认400px，高度自适应内容，背景色#FFFFFF，边框-radius 8px，阴影0 2px 8px rgba(0,0,0,0.15)

    - 标题栏：高度36px，文字#333333、14px、加粗，居左，左内边距16px，无额外边框

    - 内容区：内边距16px，表单元素（输入框、下拉框、复选框）与文字间距8px，标签文字色#333333、14px

    - 按钮区：居右摆放，包含确认/取消按钮，间距8px；确认按钮背景色#4285F4、文字#FFFFFF，取消按钮背景色#F5F5F5、文字#333333；按钮高度32px，宽度≥80px，边框-radius 4px

- 交互：点击弹窗阴影区域可关闭弹窗，按ESC键可关闭弹窗；表单操作实时反馈（如输入错误提示），点击确认按钮触发对应功能，点击取消按钮关闭弹窗

# 五、响应式与适配说明

- 窗口大小适配：调整软件窗口大小时，编辑区域、预览区内容自动换行、缩放，表格、图片不溢出窗口；侧边栏、工具栏位置固定，不随窗口缩放偏移

- 系统适配：
        

    - Windows系统：工具栏、弹窗按钮风格贴合Windows系统设计规范，字体默认微软雅黑

    - macOS系统：工具栏、弹窗按钮风格贴合macOS系统设计规范（无额外边框、圆角更柔和），字体默认苹方

    - Linux系统：适配Ubuntu/CentOS默认主题风格，确保字体显示正常、交互逻辑统一

- 异常适配：窗口缩小至最小宽度（400px）时，自动隐藏右侧预览区（分栏模式下），仅保留编辑区域，避免内容挤压错乱
